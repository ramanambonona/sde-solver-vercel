# Deploy on Vercel

This frontend is  configured for Vercel as a static site.

## Steps
1. Import the project into Vercel.
2. Keep the framework preset as **Other**.
3. No build command is required.
4. The root directory is the project root.
5. Deploy.

## Backend URL
The frontend reads the backend URL from `config.js`.

- Local: `http://localhost:8000`
- Production: `https://sde-solver-app.onrender.com`

Update `config.js` if your backend URL changes.
